# 图标文件说明

请将以下图标文件放到本目录：

- `icon.png`：32x32 托盘图标
- `32x32.png`：32x32 应用图标
- `128x128.png`：128x128 应用图标
- `128x128@2x.png`：256x256 应用图标
- `icon.ico`：Windows 图标
- `icon.icns`：macOS 图标

## 快速生成

可使用 Tauri 官方命令从一张源图自动生成：

```bash
npm run tauri icon path/to/source-image.png
```

也可以使用在线工具：<https://icon.kitchen/>
